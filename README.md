Currency Converter 💱

A simple and responsive currency converter web application that converts values between different currencies using real-time exchange rates.

🚀 Features
- Convert between multiple currencies
- Real-time exchange rates using API
- User-friendly interface
- Fast and responsive design

 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API

📂 Project Structure
- finalpro.html → Main HTML file
- finalproject.css → Styling
- finalPROJECT.js → JavaScript logic
- codes.js → Currency data

▶️ How to Run
1. Download or clone this repository
2. Open `finalpro.html` in your browser
